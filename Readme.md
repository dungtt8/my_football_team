# SaaS Football Monorepo

Hệ thống quản lý đội bóng phong trào - Monorepo Master

### Cấu trúc dự án
```
├── backend
│   ├── src
│   ├── package.json
│   └── ...
├── frontend
│   ├── src
│   ├── package.json
│   └── ...
├── package.json
└── README.md
```

### Cài đặt
1. Clone repository về máy:
```bash
git clone <repository-url>
```

2. Cài đặt các dependencies:
```bash
npm install
```

3. Chạy ứng dụng:
```bash
npm run dev
```

##### Lưu ý
- Đảm bảo bạn đã cài đặt Node.js và npm trên máy tính của mình.
- Sử dụng phiên bản Node.js tương thích với dự án để tránh lỗi không mong muốn.
